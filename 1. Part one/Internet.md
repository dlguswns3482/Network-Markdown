# Internet이란 ? 
* Internet의 Inter은 연결을 의미한다.
* `네트워크`를 많은 사람과 공유하고자 인터넷의 시작이 됨.
## Web 이란 ?
- Web에서 사용하는 www는 World Wide Web의 약자이다.
- Web처럼 인터넷이 서로 거미줄처럼 연결되어 있음.

## Internet의 특징
### `하나의 프로토콜을 사용해 통신함.`
- 프로토콜이 다르다면 통신을 하지못함.

ex) 한국어를 사용하는사람과 영어를 사용하는사람은 대화가 안됌.
- 인터넷은 TCP/IP라는 프로토콜을 사용하고있음.
